
## Synopsis

This is my React Project that is a recreation of AirBnb's Website.

## Installation

Enter The Following Commands in Terminal
 * cd desktop
 * git clone https://github.com/SamKissee/AirBNB-Clone.git project
 * cd project
 * npm install
 * npm run start
 * https://localhost:8080
